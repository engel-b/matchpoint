# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by [Keep a Changelog](https://keepachangelog.com/), and this project aims to follow Semantic Versioning.

## [Unreleased]

### Added

- VitePress documentation
- German and English documentation pages
- GitHub Pages documentation deployment
- Markdown-based project documentation
- SEO and metadata improvements
- Dependabot configuration
- Mermaid support for documentation diagrams

- Renamed GitHub Actions workflows for clearer separation of app and documentation deployment
- Improved formatting and linting setup
- Added automated formatting before commits

## [0.2.0] - 2026-07-03

### Added

- Initial public version of MatchPoint
- Progressive Web App support
- Offline-capable scoreboard
- Player profiles
- Match history
- Configurable game modes
- Touch-based scoring
- Keyboard / remote control support
- Custom key learning for input devices
- German and English app localization
- Dark and light theme
- Fullscreen support
- Undo functionality
- Local storage with IndexedDB and Dexie
- PWA icon generation from SVG
- GitHub Actions deployment to AWS S3 and CloudFront
- GitHub Release artifact generation
- Prettier, ESLint, Husky and lint-staged setup
