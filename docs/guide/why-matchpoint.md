# Warum MatchPoint?

Digitale Tischtennis-Scoreboards gibt es viele. MatchPoint verfolgt jedoch bewusst einen anderen Ansatz.

Die Anwendung wurde mit dem Ziel entwickelt, ein modernes, einfach zu bedienendes und vollständig unabhängiges Scoreboard bereitzustellen – ohne Benutzerkonto, ohne Cloud und ohne laufende Kosten.

## Warum wurde MatchPoint entwickelt?

Beim Training, Vereinsabend oder kleinen Turnieren wird häufig improvisiert: Papier, einfache Zählgeräte oder Apps, die entweder überladen sind oder eine permanente Internetverbindung voraussetzen.

MatchPoint soll genau dieses Problem lösen.

Die App konzentriert sich bewusst auf das Wesentliche: Spielstände schnell und zuverlässig erfassen – ohne Ablenkung und ohne unnötige Funktionen.

## Was unterscheidet MatchPoint von anderen Scoreboards?

MatchPoint ist keine klassische Webanwendung, sondern eine installierbare Progressive Web App.

Dadurch kombiniert sie die Vorteile einer nativen App mit der Einfachheit einer Website.

Besonderen Wert legen wir auf:

- große und gut lesbare Anzeige
- schnelle Bedienung
- Unterstützung externer Eingabegeräte
- vollständige Offline-Funktion
- lokale Datenspeicherung
- Open Source

## Warum offline-first?

Ein Tischtennis-Scoreboard sollte jederzeit funktionieren.

Ganz gleich, ob in einer Sporthalle ohne WLAN, im Keller eines Vereinsheims oder unterwegs.

Deshalb wurde MatchPoint von Anfang an als **offline-first** entwickelt.

Alle wichtigen Funktionen stehen jederzeit zur Verfügung – unabhängig von einer Internetverbindung.

## Warum Open Source?

Sport lebt von Gemeinschaft.

Dasselbe gilt für Software.

Der komplette Quellcode von MatchPoint ist öffentlich verfügbar. Jeder kann nachvollziehen, wie die Anwendung funktioniert, Fehler melden oder eigene Ideen einbringen.

Open Source schafft Transparenz, Vertrauen und ermöglicht es der Community, das Projekt gemeinsam weiterzuentwickeln.

## Warum keine Cloud?

Viele Anwendungen speichern Daten automatisch auf externen Servern.

MatchPoint geht bewusst einen anderen Weg.

Alle Daten bleiben ausschließlich auf dem eigenen Gerät.

Das hat mehrere Vorteile:

- keine Registrierung
- keine Benutzerkonten
- keine Synchronisation erforderlich
- keine Datenschutzbedenken
- vollständige Kontrolle über die eigenen Daten

Cloud-Dienste können zukünftig optional ergänzt werden. Sie werden jedoch niemals Voraussetzung für die Nutzung der App sein.

## Unsere Vision

MatchPoint soll ein modernes, zuverlässiges und freies Tischtennis-Scoreboard werden, das jeder ohne Hürden nutzen kann.

Kostenlos.

Offen.

Datenschutzfreundlich.

Und mit dem Fokus auf das Wesentliche: den Sport.
