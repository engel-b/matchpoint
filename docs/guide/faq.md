# Häufig gestellte Fragen (FAQ)

## Warum wird beim Punktezählen die Lautstärke verändert?

Einige Bluetooth-Fernbedienungen senden die Tastencodes **AudioVolumeUp** und **AudioVolumeDown** anstelle normaler Tastaturtasten.

Diese Tastencodes können von MatchPoint verwendet werden, werden jedoch gleichzeitig vom Betriebssystem verarbeitet. Dadurch verändert sich zusätzlich die Systemlautstärke.

Falls dein Eingabegerät mehrere Betriebsmodi unterstützt, empfiehlt sich ein Modus, der Standard-Tastaturtasten wie **Page Up**, **Page Down** oder Buchstaben sendet.

---

## Warum kann ich MatchPoint nicht installieren?

MatchPoint ist eine Progressive Web App (PWA). Ob eine Installation angeboten wird, hängt vom verwendeten Browser und Betriebssystem ab.

Für die beste Unterstützung empfehlen wir:

- **Android:** Google Chrome oder Microsoft Edge
- **iPhone / iPad:** Safari
- **Windows:** Microsoft Edge oder Google Chrome
- **macOS:** Safari oder Google Chrome

Stelle außerdem sicher, dass die Seite über **HTTPS** aufgerufen wird.

---

## Wo werden meine Daten gespeichert?

Alle Daten werden ausschließlich lokal auf deinem Gerät gespeichert.

Dazu gehören unter anderem:

- Spielerprofile
- Spieleinstellungen
- Spielhistorie
- Tastenbelegungen

MatchPoint verwendet dafür die Browser-Technologie **IndexedDB**.

Es werden keine Daten an externe Server übertragen.

---

## Wie kann ich alle lokalen Daten löschen?

Öffne **Einstellungen → Danger Zone → Lokale Daten löschen**.

Nach einer Bestätigung werden alle lokal gespeicherten Daten entfernt, darunter Spielerprofile, Spielhistorie, Einstellungen und Tastenzuordnungen.

Diese Aktion kann nicht rückgängig gemacht werden.

---

## Funktioniert MatchPoint ohne Internet?

Ja.

Nach der Installation als Progressive Web App kann MatchPoint vollständig offline verwendet werden.

Eine Internetverbindung wird lediglich benötigt, um die Anwendung erstmals zu laden oder Updates herunterzuladen.

---

## Kann ich meine Daten sichern?

Eine Export- und Importfunktion ist derzeit noch nicht verfügbar.

Da sämtliche Daten lokal gespeichert werden, gehen sie beim Löschen der Browserdaten oder beim Entfernen der installierten App verloren.

Eine Sicherungsfunktion ist für zukünftige Versionen geplant.

---

## Welche Browser werden unterstützt?

MatchPoint unterstützt alle modernen Browser mit Unterstützung für Progressive Web Apps und IndexedDB.

Dazu gehören unter anderem:

- Google Chrome
- Microsoft Edge
- Mozilla Firefox
- Safari

Für die Installation als App empfehlen wir Chrome, Edge oder Safari.

---

## Welche Eingabegeräte funktionieren?

Grundsätzlich funktionieren alle Geräte, die Standard-Tastatureingaben an den Browser senden.

Dazu zählen beispielsweise:

- USB-Tastaturen
- Bluetooth-Tastaturen
- HID-Fernbedienungen
- Presenter
- Bluetooth-Page-Turner
- programmierbare Keypads

Eine Übersicht getesteter Geräte findest du im Kapitel **Eingabegeräte**.

---

## Warum werden unvollständige Spiele nicht gespeichert?

MatchPoint speichert bewusst nur vollständig abgeschlossene Spiele.

Dadurch bleibt die Spielhistorie übersichtlich und enthält ausschließlich valide Ergebnisse.

Unterbrochene oder versehentlich gestartete Spiele erscheinen daher nicht in der Historie.

---

## Werden meine Daten in einer Cloud gespeichert?

Nein.

MatchPoint speichert sämtliche Daten ausschließlich lokal auf deinem Gerät.

Es erfolgt keine Synchronisation mit einer Cloud und es werden keine Benutzerkonten benötigt.

---

## Ist MatchPoint kostenlos?

Ja.

MatchPoint ist kostenlos und Open Source. Es gibt keine Werbung, keine Abonnements und keine versteckten Kosten.
