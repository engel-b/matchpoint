# Erste Schritte

Willkommen bei **MatchPoint**!

MatchPoint ist eine kostenlose Open-Source-Progressive-Web-App (PWA) zur Erfassung von Tischtennis-Spielständen. Die Anwendung wurde für den Einsatz am Tisch entwickelt und funktioniert vollständig offline.

## Voraussetzungen

MatchPoint kann auf nahezu jedem Gerät verwendet werden:

- 📱 Android-Smartphones und -Tablets
- 🍎 iPhone und iPad
- 💻 Windows
- 🍏 macOS
- 🐧 Linux

Für die Nutzung wird lediglich ein moderner Browser wie Chrome, Edge oder Safari benötigt.

## Installation

Da MatchPoint als Progressive Web App entwickelt wurde, kann die Anwendung wie eine native App installiert werden.

Die Installation bietet mehrere Vorteile:

- schneller Start direkt vom Homescreen oder Desktop
- Vollbildmodus ohne Browserleisten
- vollständige Offline-Nutzung
- automatische Updates im Hintergrund

Die Installation unterscheidet sich je nach Betriebssystem. Eine detaillierte Anleitung findest du unter **Installation**.

## Das erste Spiel

Nach dem Start der Anwendung kann sofort gespielt werden.

1. Spieler auswählen oder neue Spielerprofile anlegen.
2. Gewünschten Spielmodus festlegen (Best of 3, 5 oder 7).
3. Das Spiel durch Antippen der jeweiligen Spielerseite oder über ein konfiguriertes Eingabegerät bedienen.

Während des Spiels werden Punkte, Sätze und Aufschlag automatisch nach den offiziellen Tischtennisregeln verwaltet.

## Datenhaltung

Alle Daten werden ausschließlich lokal auf deinem Gerät gespeichert.

Dazu gehören unter anderem:

- Spielerprofile
- Spieleinstellungen
- Spielhistorie
- Tastenbelegungen

Es werden keine Benutzerkonten benötigt und keine Daten an externe Server übertragen.
