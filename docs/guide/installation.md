# Installation

MatchPoint wird als **Progressive Web App (PWA)** bereitgestellt.

Es ist keine Installation über einen App Store erforderlich. Die Anwendung wird direkt aus dem Browser installiert und anschließend wie eine normale App verwendet.

## Warum installieren?

Die Installation bietet mehrere Vorteile:

- 📲 Start direkt vom Homescreen oder Desktop
- 🌐 vollständige Offline-Nutzung
- 🚀 schneller Start ohne Browseroberfläche
- 🔄 automatische Updates im Hintergrund
- 📱 Nutzung wie eine native App

---

## Android

1. Öffne MatchPoint in **Google Chrome**.
2. Tippe auf das Menü (⋮).
3. Wähle **Zum Startbildschirm hinzufügen** oder **App installieren**.
4. Bestätige die Installation.

Anschließend erscheint MatchPoint auf dem Startbildschirm und kann wie jede andere App gestartet werden.

> **Hinweis**
>
> Je nach Android-Version oder Browser kann die Bezeichnung leicht abweichen.

---

## iPhone & iPad

1. Öffne MatchPoint in **Safari**.
2. Tippe auf **Teilen**.
3. Wähle **Zum Home-Bildschirm**.
4. Bestätige mit **Hinzufügen**.

Die Anwendung erscheint anschließend auf dem Home-Bildschirm.

> **Hinweis**
>
> Auf iOS muss Safari verwendet werden. Andere Browser unterstützen die Installation nur eingeschränkt.

---

## Windows

Unter Windows kann MatchPoint über Chromium-basierte Browser wie **Microsoft Edge** oder **Google Chrome** installiert werden.

1. Öffne MatchPoint.
2. Klicke rechts in der Adressleiste auf das Installieren-Symbol.
3. Bestätige die Installation.

Alternativ kann die Installation auch über das Browsermenü erfolgen.

---

## macOS

Unter macOS kann MatchPoint beispielsweise über **Safari** oder **Google Chrome** installiert werden.

Je nach Browser befindet sich die Installationsoption entweder in der Symbolleiste oder im Browsermenü.

---

## Nach der Installation

Beim ersten Start empfiehlt es sich,

- den gewünschten Spielmodus festzulegen,
- Spielerprofile anzulegen,
- die bevorzugte Sprache auszuwählen,
- sowie bei Bedarf externe Eingabegeräte zu konfigurieren.

Alle Einstellungen werden automatisch lokal gespeichert.

---

## Updates

Neue Versionen werden automatisch erkannt.

Ist eine neue Version verfügbar, informiert MatchPoint darüber innerhalb der Anwendung. Nach dem Bestätigen wird die neue Version geladen.

---

## Deinstallation

Da MatchPoint wie eine normale App installiert wird, kann sie jederzeit über das jeweilige Betriebssystem wieder entfernt werden.

Beim Entfernen der App werden auch alle lokal gespeicherten Daten gelöscht.
